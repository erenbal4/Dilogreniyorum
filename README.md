# Dil Öğreniyorum – Deploy Rehberi

## Adımlar

1. GitHub hesabı aç → boş repo oluştur
2. Bu klasörü ZIP olarak indir ve aç → tüm dosyaları GitHub'a yükle
3. Vercel hesabı aç → GitHub reposunu bağla
4. Ortam değişkenlerini ekle:
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
   - OPENAI_API_KEY
5. Deploy tuşuna bas → site canlı

## Kullanım

- Kayıt ol, dil ve seviye seç
- AI ile yazılı veya sesli konuşma yap
- Sertifikalarını profilinde gör
- Arkadaş ekle ve mesajlaş
- Günlük tekrar ve ilerleme takibi
